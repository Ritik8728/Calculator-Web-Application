# Calculator Web App

A modern and responsive calculator web application built using HTML, CSS, and JavaScript. The application performs basic arithmetic operations and provides an interactive user experience with keyboard support, responsive design, and real-time calculations.

## Features

- Addition, Subtraction, Multiplication, and Division
- Percentage calculation
- Positive/Negative number toggle
- Keyboard input support
- Clear and Backspace functionality
- Responsive design for desktop and mobile devices
- Error handling for invalid operations
- Interactive and modern user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Objective

The objective of this project is to develop a fully functional calculator capable of performing basic mathematical operations while providing a smooth and user-friendly interface.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the `calculator.html` file in any modern web browser.
